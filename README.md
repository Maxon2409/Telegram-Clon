# Telegram Clon

Мобильный клон Telegram на React Native (Expo).

## Возможности

- **Чаты** — список чатов с поиском, непрочитанные, онлайн-статус
- **Экран чата** — сообщения, ввод текста, галочки прочтения
- **Звонки** — история звонков (входящие/исходящие/пропущенные)
- **Контакты** — список контактов с поиском
- **Настройки** — профиль и меню настроек

## Запуск

1. Установи [Node.js](https://nodejs.org/) (LTS).
2. Установи зависимости:

```bash
npm install
```

3. Запусти приложение:

```bash
npx expo start
```

4. Открой в симуляторе или на телефоне:
   - **Android**: нажми `a` в терминале или отсканируй QR Expo Go
   - **iOS**: нажми `i` в терминале (только на Mac) или отсканируй QR в Expo Go

## Структура

```
├── app/                 # Экран (Expo Router)
│   ├── (tabs)/          # Нижние вкладки
│   │   ├── chats.tsx    # Список чатов
│   │   ├── calls.tsx    # Звонки
│   │   ├── contacts.tsx # Контакты
│   │   └── settings.tsx # Настройки
│   └── chat/[id].tsx    # Экран чата
├── constants/           # Цвета и константы
├── data/                # Мок-данные
└── types/               # TypeScript типы
```
